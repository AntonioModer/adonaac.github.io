-- bind the key 'a' to the action 'move_left' on initialization
fg.input:bind('a', 'move_left')

-- check if the action 'move_left' is being pressed on update
if fg.input:down('move_left') then
    local vx, vy = self.body:getLinearVelocity()
    self.body:setLinearVelocity(-200, vy)
end


Description

The fg.Input class handles binding keys to actions and then enables checking of actions in the update loop. The instance fg.input is automatically created when fg.init() is called, but you can also create your own instances by doing input = fg.Input. If you do this you must add input:keypressed, input:keyreleased, input:mousepressed, input:mousereleased, input:gamepadpressed, input:gamepadreleased, input:gamepadaxis to their respective love callbacks. The main.lua file that ships with fuccboiGDX already does this for the default fg.input instance (via the fg.[whatever] calls).

function love.load()
    fg.init()
    input = fg.Input()
end

function love.keypressed(key)
    fg.keypressed(key)
    input:keypressed(key)
end

function love.keyreleased(key)
    fg.keyreleased(key)
    input:keyreleased(key)
end

function love.mousepressed(x, y, button)
    fg.mousepressed(button)
    input:mousepressed(button)
end

function love.mousereleased(x, y, button)
    fg.mousereleased(button)
    input:mousereleased(button)
end

function love.gamepadpressed(joystick, button)
    fg.gamepadpressed(joystick, button)
    input:gamepadpressed(joystick, button)
end

function love.gamepadreleased(joystick, button)
    fg.gamepadreleased(joystick, button)
    input:gamepadreleased(joystick, button)
end

function love.gamepadaxis(joystick, axis, new_value)
    fg.gamepadaxis(joystick, axis, new_value)
    input:gamepadaxis(joystick, axis, new_value)
end


Methods

:bind(key[string], action[string])
  • key: the key constant, see Constants for a complete list of possible keys
  • action: the action name bound to this key 

:down(action[string])
  • returns true if the button bound to this action is down, false otherwise
  • action: the action being checked 

:pressed(action[string])
  • returns true if the button bound to this action has just been pressed, false otherwise
  • action: the action being checked 

:released(action[string])
  • returns true if the button bound to this action has just been released, false otherwise
  • action: the action being checked 

:unbind(key[string])
  • unbinds a key, and whatever action was bound to it will no longer be activated on key press/down/release
  • key: the key being unbound 

:unbindAll()
  • unbinds all keys and actions 

:update()
  • updates input state
