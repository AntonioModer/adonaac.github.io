animation:update(dt)

-- draw the animation
animation:draw(x, y)
-- the below image is the result, 'loop' mode is set by default

Description

The fg.Animation class handles the creation of animated sprites from sprite sheets. You have control over what to do with the animation itself, since it’s returned and you just hold it in a variable. The draw operation, like LÖVE’s, also takes rotation, scaling and shearing parameters after x, y. The module also supports reading sprite sheets from texture atlases on top of isolated spritesheets.

Methods

:new(image[Image], width[number], height[number], delay[number], frames[number[optional]])
  • image: the sprite sheet containing the full animation
  • width, height: the width and height of each frame
  • delay: the delay between each frame in seconds
  • frames: the number of frames to use for this animation, omitting this parameter or passing 0 tells the library to automatically handle it 

:new(image[Image], x[number], y[number], width[number], height[number], delay[number], frames[number])
  • image: the texture atlas containing the full animation and
  • x, y: the top-left position of the first animation frame
  • width, height: the width and height of each frame
  • delay: the delay between each frame in seconds
  • frames: the number of frames to use for this animation, can’t be omitted or 0 when using a texture atlas 

:update(dt[number])
  • dt: delta value passed from the main loop to update the animation 

:draw(x[number], y[number])
  • x, y: position to draw the animation, with optional values as seen in love.graphics.draw 

:addFrame(x[number], y[number], width[number], height[number], delay[number])
  • x, y: left top coordinate of the frame on the original sprite sheet
  • width, height: the width and height of each frame
  • delay: the delay between each frame in seconds 

:getHeight()
  • returns the height of the current frame 

:getWidth()
  • returns the width of the current frame 

:play()
  • starts the animation if it was stopped 

:reset()
  • goes back to the first frame 

:seek(frame[number])
  • frame: the frame position to seek, minimum is 1 and maximum is .size 

:setDelay(frame[number], delay[number])
  • frame: the frame position to set the delay
  • delay: the new delay 

:setMode(mode[string])
  • mode: the animation play mode

Possible modes:

  • 'loop': loops the animation after it’s over
  • 'once': plays the animation only once
  • 'bounce': play it once, play it in reverse, play it again (looping) 

:setSpeed(speed[number])
  • speed: the speed to play the animation at, 1 is normal, 2 is twice as fast… 

:stop()
  • stops the animation

Attributes

.current_frame[number]
  • the frame currently being displayed, minimum is 1 and maximum is .size 

.frame_height[number]
  • the height of the animation, can also be accessed through :getHeight 

.frame_width[number]
  • the width of the animation, can also be accessed through :getWidth() 

.size[number]
  • the number of frames the animation has
